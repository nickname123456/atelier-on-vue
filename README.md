# Ателье на vue

Это мой старый проект, который я переделал на фреймворк vue.


## Как заставить это работать?

### Скачать нужные пакеты
```sh
npm install
```

### Запустить фронтенд

```sh
npm run dev
```

### Запустить бэкенд

```sh
npm start
```
